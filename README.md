# hangman-game

A terminal-based Hangman game built in Python with 3 word categories,
ASCII art gallows, and full win/loss record tracking.

## How to run
hangman_game.py

## Features
- 3 categories: Technology, Animals, Countries
- ASCII art that builds with each wrong guess
- Tracks wins and losses across multiple games
- Input validation and duplicate guess detection

## What I learned
- Dictionaries and nested data structures
- Multi-stage game logic with while loops
- ASCII art stored in lists
- Functions returning values (True/False)

## Tech used
Python 3
